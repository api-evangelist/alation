# Alation GraphQL API

Alation is a data intelligence platform for data search and discovery. The API covers data source management, data objects (tables, columns, queries), catalog articles, lineage, governance policies, user endorsements, and trusted data certification.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Alation REST API
**Documentation:** https://developer.alation.com

- Reference: https://developer.alation.com/dev/reference
